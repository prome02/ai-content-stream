# AI 生成個人化內容資訊流平台

## 專案概述

打造一個「AI生成的無限資訊流」平台，讓使用者在無需對話的情況下，透過行為反饋自動生成個人化內容。

### 核心價值
- **無縫體驗**：不需要與 AI 對話，維持使用者原有的滑動習慣
- **智能學習**：根據點讚、停留時間等潛行為自動優化內容
- **無限內容**：利用 LLM 持續生成高品質的個人化內容
